**SQL_Data_Cleaning_NashvilleHousing**

**Project Overview**

This project focuses on cleaning and preparing the Nashville Housing dataset using SQL Server. The objective was to transform raw data into a clean and analysis-ready dataset by applying common data cleaning techniques.

## Skills Demonstrated

- SQL Server
- Data Cleaning
- Data Transformation
- CTEs
- Window Functions
- String Functions
- Data Standardization

## Tasks Performed

- Standardized date format
- Filled missing property addresses using self joins
- Split address columns into separate fields
- Standardized categorical values (Y/N → Yes/No)
- Identified and removed duplicate records
- Removed unused columns
